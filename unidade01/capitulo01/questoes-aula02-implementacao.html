1)
Verificar Maioridade
Básico
Crie um programa chamado VerificarIdade que declare uma variável de idade e use if/else para imprimir se a pessoa é maior ou menor de idade.
Requisitos
  • Declare int idade = 20
  • Use if (idade >= 18) para verificar a maioridade
  • Imprima "Você é maior de idade." ou "Você é menor de idade." conforme o caso
  • Imprima também a idade no final: "Idade informada: 20"
Solução
public class VerificarIdade { public static void main(String[] args) { int idade = 20; if (idade >= 18) { System.out.println("Você é maior de idade."); } else { System.out.println("Você é menor de idade."); } System.out.println("Idade informada: " + idade); } }
Saída esperada no console
Você é maior de idade. Idade informada: 20
2)
Classificar Nota com Conceito
Básico
Crie um programa chamado ClassificarNota que use if / else if / else para atribuir um conceito a uma nota de 0 a 100.
Requisitos
  • Declare int nota = 85
  • Use if / else if / else: nota >= 90 → "A", nota >= 70 → "B", nota >= 50 → "C", senão → "D"
  • Imprima a nota e o conceito no formato "Nota: 85 | Conceito: B"
Solução
public class ClassificarNota { public static void main(String[] args) { int nota = 85; String conceito; if (nota >= 90) { conceito = "A"; } else if (nota >= 70) { conceito = "B"; } else if (nota >= 50) { conceito = "C"; } else { conceito = "D"; } System.out.println("Nota: " + nota + " | Conceito: " + conceito); } }
Saída esperada no console
Nota: 85 | Conceito: B
Dica: As condições são avaliadas em ordem. Como 85 não é >= 90, a primeira condição falha. Como 85 é >= 70, a segunda condição é verdadeira e executa — as demais são ignoradas.
3)
Dias da Semana com switch/case
Básico
Crie um programa chamado DiasDaSemana que use switch/case para converter um número (1–7) no nome do respectivo dia da semana.
Requisitos
  • Declare int dia = 3
  • Use switch(dia) com cases para 1 (Domingo) até 7 (Sábado)
  • Cada case deve ter break ao final
  • O default deve imprimir "Número inválido."
  • Imprima no formato "Dia 3: Terça-feira"
Solução
public class DiasDaSemana { public static void main(String[] args) { int dia = 3; String nomeDia; switch (dia) { case 1: nomeDia = "Domingo"; break; case 2: nomeDia = "Segunda-feira"; break; case 3: nomeDia = "Terça-feira"; break; case 4: nomeDia = "Quarta-feira"; break; case 5: nomeDia = "Quinta-feira"; break; case 6: nomeDia = "Sexta-feira"; break; case 7: nomeDia = "Sábado"; break; default: nomeDia = "Número inválido."; } System.out.println("Dia " + dia + ": " + nomeDia); } }
Saída esperada no console
Dia 3: Terça-feira
4)
Contador e Soma com for
Básico
Crie um programa chamado ContadorSoma que use um loop for para imprimir os números de 1 a 5 e calcular a soma total.
Requisitos
  • Use for (int i = 1; i <= 5; i++)
  • Dentro do loop, imprima cada número e acumule em uma variável soma
  • Após o loop, imprima "Soma total: 15"
Solução
public class ContadorSoma { public static void main(String[] args) { int soma = 0; for (int i = 1; i <= 5; i++) { System.out.println("Número: " + i); soma += i; } System.out.println("Soma total: " + soma); } }
Saída esperada no console
Número: 1 Número: 2 Número: 3 Número: 4 Número: 5 Soma total: 15
Dica: Note que o loop começa em i = 1 e usa i <= 5 (menor ou igual). Isso faz o loop rodar para 1, 2, 3, 4, 5 — diferente de i < 5 que pararia em 4.
5)
Tabuada com for
Básico–Médio
Crie um programa chamado Tabuada que gere a tabuada completa de um número usando um loop for, imprimindo cada linha no formato "N x i = resultado".
Requisitos
  • Declare int numero = 7
  • Imprima o cabeçalho "Tabuada do 7:"
  • Use for (int i = 1; i <= 10; i++) para gerar as multiplicações
  • Imprima cada linha no formato "7 x 1 = 7", "7 x 2 = 14", etc.
Solução
public class Tabuada { public static void main(String[] args) { int numero = 7; System.out.println("Tabuada do " + numero + ":"); for (int i = 1; i <= 10; i++) { System.out.println(numero + " x " + i + " = " + (numero * i)); } } }
Saída esperada no console
Tabuada do 7: 7 x 1 = 7 7 x 2 = 14 7 x 3 = 21 7 x 4 = 28 7 x 5 = 35 7 x 6 = 42 7 x 7 = 49 7 x 8 = 56 7 x 9 = 63 7 x 10 = 70
Conceitos aplicados: O loop for com variável i de 1 a 10 combina com o operador * para calcular o produto. Os parênteses em (numero * i) garantem que a multiplicação seja feita antes da concatenação com o +.
