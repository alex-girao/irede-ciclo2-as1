Perfil do Usuário
Básico
Crie um programa Java chamado PerfilUsuario que declare variáveis para armazenar as informações de uma pessoa e as exiba no console. Use o tipo de dado mais adequado para cada informação.
Requisitos
  • Declare uma variável String para o nome da pessoa (ex: "Maria")
  • Declare uma variável int para a idade (ex: 22)
  • Declare uma variável double para a altura em metros (ex: 1.68)
  • Declare uma variável boolean indicando se é estudante (ex: true)
  • Declare uma variável char para a inicial do nome (ex: 'M')
  • Imprima cada variável com uma mensagem descritiva usando System.out.println()
Solução
public class PerfilUsuario { public static void main(String[] args) { String nome = "Maria"; int idade = 22; double altura = 1.68; boolean ehEstudante = true; char inicial = 'M'; System.out.println("Nome: " + nome); System.out.println("Idade: " + idade + " anos"); System.out.println("Altura: " + altura + " m"); System.out.println("Estudante: " + ehEstudante); System.out.println("Inicial: " + inicial); } }
Saída esperada no console
Nome: Maria Idade: 22 anos Altura: 1.68 m Estudante: true Inicial: M

Calculadora de Operadores
Básico
Crie um programa Java chamado CalculadoraBasica que declare dois inteiros e aplique todos os operadores aritméticos sobre eles, exibindo cada resultado com a operação correspondente.
Requisitos
  • Declare dois inteiros: a = 15 e b = 4
  • Calcule e imprima a adição a + b
  • Calcule e imprima a subtração a - b
  • Calcule e imprima a multiplicação a * b
  • Calcule e imprima a divisão inteira a / b (observe o resultado!)
  • Calcule e imprima o módulo (resto) a % b
Solução
public class CalculadoraBasica { public static void main(String[] args) { int a = 15; int b = 4; System.out.println("Adição: " + (a + b)); // 19 System.out.println("Subtração: " + (a - b)); // 11 System.out.println("Multiplicação: " + (a * b)); // 60 System.out.println("Divisão: " + (a / b)); // 3 (int descarta a parte decimal) System.out.println("Módulo: " + (a % b)); // 3 (resto de 15 ÷ 4) } }
Saída esperada no console
Adição: 19 Subtração: 11 Multiplicação: 60 Divisão: 3 Módulo: 3
3)
Área e Perímetro do Retângulo
Básico–Médio
Crie um programa Java chamado Retangulo que calcule a área e o perímetro de um retângulo, dados sua largura e altura, e exiba os resultados formatados.
Requisitos
  • Declare double largura = 5.5 e double altura = 3.2
  • Calcule a área com a fórmula: area = largura * altura
  • Calcule o perímetro com a fórmula: perimetro = 2 * (largura + altura)
  • Imprima as duas dimensões, a área (em m²) e o perímetro (em m)
  • Armazene os resultados intermediários em variáveis antes de imprimir
Solução
public class Retangulo { public static void main(String[] args) { double largura = 5.5; double altura = 3.2; double area = largura * altura; double perimetro = 2 * (largura + altura); System.out.println("Largura: " + largura + " m"); System.out.println("Altura: " + altura + " m"); System.out.println("Área: " + area + " m²"); System.out.println("Perímetro: " + perimetro + " m"); } }
Saída esperada no console
Largura: 5.5 m Altura: 3.2 m Área: 17.6 m² Perímetro: 17.4 m

Conversor de Temperatura
Médio
Crie um programa Java chamado ConversorTemperatura que converta uma temperatura em Celsius para Fahrenheit e Kelvin, aplicando as fórmulas de conversão corretas e exibindo os três valores.
Requisitos
  • Declare double celsius = 25.0
  • Converta para Fahrenheit com a fórmula: F = (C * 9.0 / 5.0) + 32
  • Converta para Kelvin com a fórmula: K = C + 273.15
  • Armazene cada resultado em uma variável separada antes de imprimir
  • Imprima as três temperaturas com suas unidades (°C, °F, K)
  • Atenção: use 9.0 (e não 9) para garantir divisão decimal!
Solução
public class ConversorTemperatura { public static void main(String[] args) { double celsius = 25.0; double fahrenheit = (celsius * 9.0 / 5.0) + 32; double kelvin = celsius + 273.15; System.out.println("Celsius: " + celsius + " °C"); System.out.println("Fahrenheit: " + fahrenheit + " °F"); System.out.println("Kelvin: " + kelvin + " K"); } }
Saída esperada no console
Celsius: 25.0 °C Fahrenheit: 77.0 °F Kelvin: 298.15 K
Dica: Se você escrever 9 / 5 em vez de 9.0 / 5.0, o Java fará divisão inteira e o resultado será 1, produzindo Fahrenheit errado. Sempre que precisar de resultado decimal numa divisão, certifique-se de que pelo menos um dos operandos seja double ou float.

Cálculo de Salário Líquido
Desafio
Crie um programa Java chamado CalculoSalario que simule o cálculo de um holerite simplificado. Dado um salário bruto, calcule os descontos obrigatórios (INSS e IRRF), os benefícios e o salário líquido final, usando múltiplos tipos de variáveis e expressões encadeadas.
Dados de entrada (declare como variáveis)
  • double salarioBase = 3500.00
  • double taxaINSS = 0.075 — 7,5% do salário bruto
  • double taxaIRRF = 0.15 — 15% do salário bruto (simplificado)
  • double valeRefeicao = 550.00
  • double valeTransporte = 150.00
Cálculos a realizar
  • Calcule o desconto do INSS: salarioBase * taxaINSS
  • Calcule o desconto do IRRF: salarioBase * taxaIRRF
  • Calcule o total de descontos: soma dos dois descontos
  • Calcule o salário líquido: salarioBase - totalDescontos + valeRefeicao + valeTransporte
  • Declare um boolean recebeBonus = salarioLiquido < 4000 usando operador relacional
  • Imprima todos os valores organizados como um holerite
Solução
public class CalculoSalario { public static void main(String[] args) { // Dados de entrada double salarioBase = 3500.00; double taxaINSS = 0.075; double taxaIRRF = 0.15; double valeRefeicao = 550.00; double valeTransporte = 150.00; // Cálculos double descontoINSS = salarioBase * taxaINSS; double descontoIRRF = salarioBase * taxaIRRF; double totalDescontos = descontoINSS + descontoIRRF; double salarioLiquido = salarioBase - totalDescontos + valeRefeicao + valeTransporte; boolean recebeBonus = salarioLiquido < 4000; // Saída System.out.println("===== HOLERITE ====="); System.out.println("Salário Bruto: R$ " + salarioBase); System.out.println("Desconto INSS: R$ " + descontoINSS); System.out.println("Desconto IRRF: R$ " + descontoIRRF); System.out.println("Total Descontos: R$ " + totalDescontos); System.out.println("Vale Refeição: R$ " + valeRefeicao); System.out.println("Vale Transporte: R$ " + valeTransporte); System.out.println("===================="); System.out.println("Salário Líquido: R$ " + salarioLiquido); System.out.println("Elegível p/ Bônus: " + recebeBonus); } }
Saída esperada no console
===== HOLERITE ===== Salário Bruto: R$ 3500.0 Desconto INSS: R$ 262.5 Desconto IRRF: R$ 525.0 Total Descontos: R$ 787.5 Vale Refeição: R$ 550.0 Vale Transporte: R$ 150.0 ==================== Salário Líquido: R$ 3412.5 Elegível p/ Bônus: true
Conceitos aplicados: Esta questão combina double para valores monetários, boolean com operador relacional <, múltiplas expressões aritméticas encadeadas e organização da saída. O operador recebeBonus = salarioLiquido < 4000 demonstra que operadores relacionais retornam diretamente um boolean.
