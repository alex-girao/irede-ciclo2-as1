O encapsulamento é a prática de ocultar os detalhes internos de uma classe, expondo apenas o que é necessário para quem a utiliza.
02
O modificador public permite que um atributo seja acessado somente dentro da própria classe.
03
Atributos declarados como private não podem ser acessados diretamente de fora da classe.
04
Os métodos Getter e Setter são usados para acessar e modificar atributos privados de forma controlada.
05
Por convenção, um método Getter tem o prefixo get seguido do nome do atributo com a primeira letra maiúscula. Ex: getSaldo().
06
Um método Setter em Java retorna o valor do atributo que está modificando.
07
Um construtor em Java pode ter qualquer nome, desde que seja seguido de parênteses.
08
Um construtor em Java não possui tipo de retorno — nem mesmo void.
09
A sobrecarga de construtores permite que uma classe tenha múltiplos construtores com diferentes listas de parâmetros.
10
Se nenhum construtor for declarado em uma classe, o Java cria automaticamente um construtor padrão sem parâmetros.
