Percorrer Array com for
Básico
Crie um programa Java chamado PercorrerArray que declare um array de notas e percorra-o com um laço for, imprimindo cada elemento com seu respectivo índice.
Requisitos
  • Declare int[] notas = {7, 9, 5, 8, 10}
  • Use um laço for com índice para percorrer o array
  • Em cada iteração, imprima o índice e o valor no formato Nota[i]: valor
  • Após o laço, imprima o total de notas usando notas.length
Solução
public class PercorrerArray { public static void main(String[] args) { int[] notas = {7, 9, 5, 8, 10}; for (int i = 0; i < notas.length; i++) { System.out.println("Nota[" + i + "]: " + notas[i]); } System.out.println("Total de notas: " + notas.length); } }
Saída esperada no console
Nota[0]: 7 Nota[1]: 9 Nota[2]: 5 Nota[3]: 8 Nota[4]: 10 Total de notas: 5

Calcular Média de um Array
Básico
Crie um programa Java chamado MediaNotas que some todos os elementos de um array de notas usando um laço e calcule a média, exibindo a soma e o resultado final.
Requisitos
  • Declare int[] notas = {6, 8, 7, 9, 10}
  • Declare int soma = 0 antes do laço
  • Use um for para acumular a soma de todos os elementos
  • Calcule a média: double media = (double) soma / notas.length
  • Imprima a soma e a média
Solução
public class MediaNotas { public static void main(String[] args) { int[] notas = {6, 8, 7, 9, 10}; int soma = 0; for (int i = 0; i < notas.length; i++) { soma += notas[i]; } double media = (double) soma / notas.length; System.out.println("Soma: " + soma); System.out.println("Média: " + media); } }
Saída esperada no console
Soma: 40 Média: 8.0
Dica: O cast (double) soma é necessário porque ambos soma e notas.length são int. Sem o cast, Java faria divisão inteira e 40 / 5 retornaria 8 (int), não 8.0 (double).

Lista de Compras com ArrayList
Básico
Crie um programa Java chamado ListaCompras que gerencie uma lista de compras usando ArrayList: adicione itens, remova um e exiba a lista final com o total de itens.
Requisitos
  • Importe java.util.ArrayList e crie ArrayList<String> compras
  • Adicione 4 itens com add(): "Leite", "Pão", "Ovos", "Queijo"
  • Remova "Ovos" pelo índice com remove(2)
  • Imprima o total de itens com size()
  • Percorra a lista com for-each e imprima cada item com "- " na frente
Solução
import java.util.ArrayList; public class ListaCompras { public static void main(String[] args) { ArrayList<String> compras = new ArrayList<>(); compras.add("Leite"); compras.add("Pão"); compras.add("Ovos"); compras.add("Queijo"); compras.remove(2); // remove "Ovos" (índice 2) System.out.println("Itens na lista: " + compras.size()); for (String item : compras) { System.out.println("- " + item); } } }
Saída esperada no console
Itens na lista: 3 - Leite - Pão - Queijo

Analisando uma String
Básico
Crie um programa Java chamado AnalisarString que declare uma String com um nome completo e aplique diferentes métodos, exibindo os resultados de cada operação.
Requisitos
  • Declare String nome = "Maria da Silva"
  • Imprima o tamanho com length()
  • Imprima o nome em maiúsculas com toUpperCase()
  • Imprima o nome em minúsculas com toLowerCase()
  • Verifique e imprima se contém "Silva" com contains()
  • Imprima o primeiro caractere com charAt(0)
  • Imprima os 5 primeiros caracteres com substring(0, 5)
Solução
public class AnalisarString { public static void main(String[] args) { String nome = "Maria da Silva"; System.out.println("Nome: " + nome); System.out.println("Tamanho: " + nome.length()); System.out.println("Maiúsculas: " + nome.toUpperCase()); System.out.println("Minúsculas: " + nome.toLowerCase()); System.out.println("Contém \"Silva\": " + nome.contains("Silva")); System.out.println("1º caractere: " + nome.charAt(0)); System.out.println("Primeiros 5: " + nome.substring(0, 5)); } }
Saída esperada no console
Nome: Maria da Silva Tamanho: 14 Maiúsculas: MARIA DA SILVA Minúsculas: maria da silva Contém "Silva": true 1º caractere: M Primeiros 5: Maria

Processar Data com split()
Básico–Médio
Crie um programa Java chamado ProcessarData que receba uma data no formato "AAAA-MM-DD", use split() para separar suas partes e exiba o ano, mês e dia individualmente — além de remontar a data no formato brasileiro "DD/MM/AAAA".
Requisitos
  • Declare String data = "2025-05-26"
  • Use split("-") para dividir a data em um array String[] partes
  • Armazene as partes em variáveis: ano = partes[0], mes = partes[1], dia = partes[2]
  • Imprima a data original, o ano, o mês e o dia separadamente
  • Monte e imprima a data no formato brasileiro usando concatenação: dia + "/" + mes + "/" + ano
Solução
public class ProcessarData { public static void main(String[] args) { String data = "2025-05-26"; String[] partes = data.split("-"); String ano = partes[0]; String mes = partes[1]; String dia = partes[2]; System.out.println("Data original: " + data); System.out.println("Ano: " + ano); System.out.println("Mês: " + mes); System.out.println("Dia: " + dia); System.out.println("Formatada: " + dia + "/" + mes + "/" + ano); } }
Saída esperada no console
Data original: 2025-05-26 Ano: 2025 Mês: 05 Dia: 26 Formatada: 26/05/2025
Conceitos aplicados: split() retorna um String[] — ou seja, um array de Strings. Combinamos aqui o conhecimento de arrays (acesso por índice), Strings (split e concatenação com +) e a leitura de partes individuais de um dado estruturado.
