O que é Java? Linguagem de Programação

Java é uma linguagem de programação de alto nível, orientada a objetos e baseada em classes, amplamente utilizada no desenvolvimento de aplicações para diversas plataformas.

Criada por James Gosling e sua equipe na Sun Microsystems — posteriormente adquirida pela Oracle.

Primeiro Programa: Olá, Mundo! HelloWorld

O objetivo do primeiro programa é imprimir "Olá, mundo!" no console.

public class HelloWorld { public static void main(String[] args) { System.out.println("Olá, mundo!"); } }

System.out.println() imprime texto e pula uma linha. O método main é o ponto de entrada de qualquer programa Java.

java.util.Scanner Entrada de Dados

O Scanner é uma classe da biblioteca padrão do Java usada para ler dados digitados pelo usuário no console. Para utilizá-la, dois passos são obrigatórios: importar a classe no topo do arquivo e instanciar um objeto Scanner apontando para System.in (a entrada padrão do teclado).

import java.util.Scanner; // ① importar antes da declaração da classe public class MinhaClasse { public static void main(String[] args) { Scanner scanner = new Scanner(System.in); // ② criar o Scanner // ... ler dados aqui ... scanner.close(); // ③ fechar ao terminar (boa prática) } }

O nome "Java" vem do café
A equipe escolheu "Java" em referência ao café produzido na ilha de Java, na Indonésia — bebida popular entre programadores. Por isso o logotipo é uma xícara de café fumegante! O nome anterior do projeto era "Oak" (carvalho).

Write Once, Run Anywhere (WORA)
O princípio central do Java: você escreve o código uma vez e ele roda em qualquer SO que tenha uma JVM. O compilador gera bytecode — um formato intermediário que a JVM de cada plataforma sabe executar.
📱
Android é baseado em Java
Originalmente, todo desenvolvimento Android era em Java. Hoje o Kotlin é preferido, mas roda na mesma JVM e é 100% interoperável com Java. Se você aprender Java, estará a um passo do desenvolvimento mobile.

Java roda nos sistemas bancários do Brasil
Grande parte das transações financeiras no Brasil passa por sistemas escritos em Java. O Banco Central, grandes bancos e fintechs usam Java em seus sistemas críticos. Confiabilidade e robustez são os motivos principais.
🎮
Minecraft foi escrito em Java
O Minecraft — um dos jogos mais vendidos de todos os tempos — foi criado em Java por Markus "Notch" Persson. A versão Java Edition ainda existe e é muito popular entre modders. Prova de que Java não serve só para sistemas corporativos!
🔢
Por que int tem 32 bits e não 30 ou 40?
Computadores trabalham em base 2. Potências de 2 são os tamanhos naturais de memória: 8, 16, 32, 64 bits. Com 32 bits temos 2³² combinações — mais de 4 bilhões de valores. É o equilíbrio ideal entre tamanho e alcance para a maioria dos casos.

Controle de Fluxo Fundamento

Programas em Java são compostos por instruções executadas sequencialmente. Muitas vezes precisamos tomar decisões e repetir certas ações de forma controlada — é aí que entram as estruturas de controle de fluxo.

Estruturas de Decisão Estruturas de Repetição
if / else Decisão Simples

A estrutura if/else toma decisões com base em condições. A lógica: "se a condição for verdadeira, faça algo; caso contrário, faça outra coisa".

if (condição) { // executado se a condição for VERDADEIRA } else { // executado se a condição for FALSA }

Exemplo: verificar maioridade.

int idade = 20; if (idade >= 18) { System.out.println("Você é maior de idade."); } else { System.out.println("Você é menor de idade."); }
if / else if / else Decisão Encadeada

Quando há múltiplas condições a verificar, usamos else if. As condições são testadas em ordem — a primeira verdadeira executa e as demais são ignoradas.

int nota = 85; if (nota >= 90) { System.out.println("Conceito A"); } else if (nota >= 70) { System.out.println("Conceito B"); // ← executado: 85 >= 70 } else { System.out.println("Conceito C"); }
switch / case Seleção Múltipla

O switch/case substitui múltiplos if/else para casos específicos e discretos. É mais legível quando há muitas opções fixas a comparar.

ElementoFunção
switchAvalia a expressão e busca um case correspondente
caseDefine um valor específico a comparar
breakEncerra o switch — sem ele, a execução "cai" no próximo case
defaultExecutado quando nenhum case corresponde
int dia = 3; switch (dia) { case 1: System.out.println("Domingo"); break; case 2: System.out.println("Segunda"); break; case 3: System.out.println("Terça"); // ← executado: dia == 3 break; default: System.out.println("Outro dia"); }
Estruturas de Repetição Loops

As estruturas de repetição são utilizadas para executar um bloco de código várias vezes, enquanto uma condição específica for verdadeira. Java possui três tipos principais:

for
Iterações fixas
Quando sabe quantas vezes repetir
while
Pré-teste
Testa antes de executar
do-while
Pós-teste
Executa ao menos uma vez
Loop for Iterações Definidas

Utilizado quando se sabe exatamente quantas vezes o loop será executado. Concentra inicialização, condição e incremento em uma única linha.

for (int i = 0; i < 5; i++) { System.out.println("Valor de i: " + i); } // Saída: 0, 1, 2, 3, 4
ParteCódigoDescrição
Inicializaçãoint i = 0Executada 1× no início
Condiçãoi < 5Verificada antes de cada iteração
Incrementoi++Executado após cada iteração
Loop while Pré-Teste

Utilizado quando não se sabe o número exato de repetições. A condição é verificada antes de cada execução — se já for falsa no início, o bloco nunca executa.

int contador = 0; while (contador < 5) { System.out.println("Contador: " + contador); contador++; // ESSENCIAL: sem isso, loop infinito! } // Saída: 0, 1, 2, 3, 4
Loop do-while Pós-Teste

Semelhante ao while, mas garante que o bloco seja executado pelo menos uma vez. A condição é verificada depois da execução.

int numero = 0; do { System.out.println("Número: " + numero); numero++; } while (numero < 3); // Saída: 0, 1, 2

Mesmo que a condição já seja falsa desde o início, o bloco do executa uma vez antes de verificar.

Curiosidades sobre Controle de Fluxo 🧠
🧠
if/else existe desde os anos 1950
A estrutura if/else foi introduzida na linguagem FORTRAN em 1957, uma das primeiras linguagens de alto nível. Desde então tornou-se parte fundamental de praticamente todas as linguagens de programação do mundo.
⚠
Cuidado com o "fall-through" do switch
Sem o break, a execução "cai" para o próximo case — comportamento chamado fall-through. Isso pode causar bugs sutis e difíceis de encontrar. Em Java 14+, o switch expression com -> elimina esse problema automaticamente.
♾
O perigo do loop infinito
Um while sem condição de parada válida roda para sempre e trava o programa — isso é chamado de loop infinito. Sempre garanta que a condição eventualmente se tornará false, normalmente incrementando um contador ou alterando uma variável de controle.
⚡
for-each — o loop mais moderno
Java possui uma variante do for chamada for-each (enhanced for), usada para percorrer arrays e coleções: for (int n : numeros) { ... }. É mais legível e evita erros de índice. Você aprenderá isso nas próximas aulas!
🤔
switch vs if/else — quando usar cada um?
Use switch quando comparar a mesma variável com valores fixos e discretos (dias da semana, opções de menu). Use if/else para intervalos (>=, <), condições compostas (&&, ||) ou tipos não suportados pelo switch, como boolean.
🔄
do-while é ideal para menus interativos
O do-while é perfeito para menus de console: o menu precisa ser exibido pelo menos uma vez, e continua aparecendo enquanto o usuário não escolher "Sair". É um dos poucos cenários onde o pós-teste é claramente superior ao pré-teste.

O que é um Array? Estrutura de Dados

Um array é uma estrutura de dados que armazena uma coleção indexada de elementos do mesmo tipo. Útil quando precisamos trabalhar com múltiplos dados de forma organizada, sem declarar diversas variáveis separadas.

Mesmo Tipo Tamanho Fixo Indexado a partir do 0
Declaração e Acesso Sintaxe

Um array é declarado com tipo[] e os elementos são acessados pelo índice entre colchetes. O primeiro elemento sempre está na posição índice 0 e o último em length - 1.

// Declaração com valores iniciais int[] numeros = {10, 25, 37, 42, 56}; // Acessando elementos pelo índice System.out.println(numeros[0]); // 10 — primeiro elemento System.out.println(numeros[4]); // 56 — último elemento System.out.println(numeros.length); // 5 — tamanho do array // Declaração com tamanho fixo (valores inicializados em 0) int[] notas = new int[5];
Percorrendo Arrays for e for-each

Use o laço for com índice quando precisar da posição, ou o for-each para percorrer todos os elementos de forma mais simples.

int[] notas = {7, 9, 5, 8, 10}; // for com índice — acessa por posição for (int i = 0; i < notas.length; i++) { System.out.println("Nota " + i + ": " + notas[i]); } // for-each — mais simples, sem acesso ao índice for (int nota : notas) { System.out.println(nota); }
ArrayList — Arrays Dinâmicos java.util

Arrays tradicionais têm tamanho fixo. O ArrayList redimensiona automaticamente, sendo ideal quando o número de elementos não é conhecido antecipadamente.

import java.util.ArrayList; ArrayList<Integer> numeros = new ArrayList<>(); numeros.add(1); // adiciona ao final numeros.add(2); numeros.add(3); System.out.println(numeros.get(0)); // 1 — acessa por índice numeros.set(1, 5); // substitui posição 1 por 5 numeros.remove(0); // remove elemento no índice 0 System.out.println(numeros.size()); // tamanho atual da lista for (int num : numeros) { System.out.println(num); }

Arrays Multidimensionais Matrizes

Um array bidimensional funciona como uma tabela de linhas e colunas. O acesso usa dois índices: matriz[linha][coluna].

int[][] matriz = { {1, 2, 3}, {4, 5, 6}, {7, 8, 9} }; System.out.println(matriz[0][0]); // 1 — linha 0, coluna 0 System.out.println(matriz[1][2]); // 6 — linha 1, coluna 2 System.out.println(matriz[2][1]); // 8 — linha 2, coluna 1
String em Java Classe especial

Em Java, String é uma classe usada para representar texto. Strings são objetos imutáveis — cada operação que parece alterá-la na verdade cria um novo objeto na memória.

// Forma 1 — literal (mais comum e eficiente) String nome = "Maria"; // Forma 2 — via construtor (evite quando possível) String sobrenome = new String("Silva");

Prefira sempre a Forma 1 — o Java reutiliza strings literais de um pool de memória compartilhado, consumindo menos recursos.

Curiosidades sobre Arrays e Strings 🧵
0
Por que arrays começam no índice 0?
O índice 0 tem origem na aritmética de ponteiros: o índice representa o deslocamento em relação ao início do array na memória. O primeiro elemento está a "0 posições" do início. O matemático Dijkstra também argumentou que começar em 0 torna as expressões de faixa (0 ≤ i < n) mais elegantes e menos propensas a erros.
🔒
Strings são imutáveis — mas por quê?
A imutabilidade de String em Java é uma decisão de design intencional: permite que a JVM reutilize objetos String em um String Pool, economizando memória. Também torna Strings thread-safe por natureza — múltiplas threads podem compartilhar a mesma string sem risco de corrupção de dados.
📈
ArrayList dobra de tamanho automaticamente
Internamente, o ArrayList usa um array comum. Quando esse array fica cheio, o ArrayList cria um novo array com o dobro da capacidade e copia todos os elementos. Isso acontece de forma transparente. Por isso, é recomendado inicializar o ArrayList com uma capacidade estimada quando o número de elementos for conhecido: new ArrayList<>(100).
⚠
Nunca compare Strings com == em Java
O operador == compara referências de memória, não o conteúdo. Duas strings com o mesmo texto podem estar em endereços de memória diferentes. Use sempre equals() para comparar conteúdo: a.equals(b). É um dos erros mais comuns de iniciantes em Java!
⚡
StringBuilder é muito mais rápido que String +
Como strings são imutáveis, cada uso de + cria um novo objeto. Em loops com muitas concatenações, isso gera um enorme consumo de memória. A classe StringBuilder resolve isso acumulando as partes antes de criar a string final: sb.append("texto"). Em 10.000 concatenações, pode ser 100x mais rápido.
📊
String é a classe mais usada em Java
Praticamente todo programa Java manipula strings — nomes, mensagens, arquivos, URLs, JSON. A classe String possui mais de 60 métodos na sua API. Além dos métodos de instância, a classe String também oferece métodos estáticos úteis como String.format() para formatar texto com precisão.
