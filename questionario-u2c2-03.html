1)
CriarSuperclasse
Básico
Crie a classe Veiculo — ela será a superclasse das próximas questões. Ela deve ter dois atributos privados e um método que exibe as informações no console.
Requisitos
  • Declare os atributos private String marca e private int velocidadeMax
  • Crie um construtor que receba marca e velocidadeMax como parâmetros
  • Crie getters: getMarca() e getVelocidadeMax()
  • Crie o método exibirInfo() que imprime marca e velocidade máxima
Solução
public class Veiculo { private String marca; private int velocidadeMax; public Veiculo(String marca, int velocidadeMax) { this.marca = marca; this.velocidadeMax = velocidadeMax; } public String getMarca() { return marca; } public int getVelocidadeMax() { return velocidadeMax; } public void exibirInfo() { System.out.println("Marca: " + marca); System.out.println("Vel. máx: " + velocidadeMax + " km/h"); } }
Saída esperada (ao chamar exibirInfo)
Marca: Toyota Vel. máx: 180 km/h
2)
CriarSubclasse
Básico
Crie a classe Carro que herda de Veiculo. Além dos atributos herdados, ela deve ter um atributo próprio para o número de portas.
Requisitos
  • Use extends Veiculo para declarar a herança
  • Adicione o atributo próprio private int numPortas
  • Crie o construtor de Carro recebendo marca, velocidadeMax e numPortas
  • Use super(marca, velocidadeMax) na primeira linha do construtor
  • Crie o getter getNumPortas()
Solução
public class Carro extends Veiculo { private int numPortas; public Carro(String marca, int velocidadeMax, int numPortas) { super(marca, velocidadeMax); // 1º: inicializa Veiculo this.numPortas = numPortas; } public int getNumPortas() { return numPortas; } }
Testando no main
Carro c = new Carro("Toyota", 180, 4); System.out.println(c.getMarca()); // Toyota (herdado) System.out.println(c.getNumPortas()); // 4 (próprio)
3)
SobrescreverExibirInfo
Básico
Adicione na classe Carro a sobrescrita do método exibirInfo(). O método deve primeiro chamar a versão da superclasse e depois exibir o número de portas.
Requisitos
  • Use a anotação @Override antes da declaração do método
  • Chame super.exibirInfo() na primeira linha para exibir marca e velocidade
  • Em seguida imprima: "Portas: " + numPortas
Solução (adicionar dentro da classe Carro)
@Override public void exibirInfo() { super.exibirInfo(); // chama Veiculo.exibirInfo() System.out.println("Portas: " + numPortas); }
Saída ao chamar c.exibirInfo()
Marca: Toyota Vel. máx: 180 km/h Portas: 4
4)
CriarOutraSubclasse
Básico
Crie a classe Caminhao que também herda de Veiculo. Ela deve ter seu próprio atributo de capacidade de carga e sobrescrever exibirInfo().
Requisitos
  • Declare private double capacidadeToneladas
  • Construtor com super(marca, velocidadeMax) + inicialização da carga
  • Getter getCapacidadeToneladas()
  • Sobrescreva exibirInfo() com @Override chamando super.exibirInfo() e exibindo a capacidade
Solução
public class Caminhao extends Veiculo { private double capacidadeToneladas; public Caminhao(String marca, int velocidadeMax, double capacidadeToneladas) { super(marca, velocidadeMax); this.capacidadeToneladas = capacidadeToneladas; } public double getCapacidadeToneladas() { return capacidadeToneladas; } @Override public void exibirInfo() { super.exibirInfo(); System.out.println("Carga: " + capacidadeToneladas + " t"); } }
Saída esperada
Marca: Volvo Vel. máx: 120 km/h Carga: 15.0 t
5)
TestarHerancaNoMain
Básico
Escreva um método main que cria um Carro e um Caminhao e chama exibirInfo() em cada um, demonstrando o polimorfismo.
Requisitos
  • Crie Carro c = new Carro("Honda", 200, 4)
  • Crie Caminhao t = new Caminhao("Volvo", 120, 15.0)
  • Chame c.exibirInfo() e t.exibirInfo()
  • Observe que cada objeto chama sua própria versão do método (polimorfismo)
Solução
public static void main(String[] args) { Carro c = new Carro("Honda", 200, 4); Caminhao t = new Caminhao("Volvo", 120, 15.0); System.out.println("=== Carro ==="); c.exibirInfo(); System.out.println("\n=== Caminhão ==="); t.exibirInfo(); }
Saída esperada
=== Carro === Marca: Honda Vel. máx: 200 km/h Portas: 4 === Caminhão === Marca: Volvo Vel. máx: 120 km/h Carga: 15.0 t
6)
UsarInstanceOf
Intermediário
Escreva um trecho de código que usa o operador instanceof para verificar o tipo de objetos na hierarquia Veiculo → Carro.
Requisitos
  • Crie um objeto Carro c = new Carro("Fiat", 160, 2)
  • Verifique com instanceof: c é um Carro? c é um Veiculo? c é um Caminhao?
  • Imprima o resultado de cada verificação
  • Observe que instanceof considera toda a hierarquia
Solução
Carro c = new Carro("Fiat", 160, 2); System.out.println(c instanceof Carro); // true System.out.println(c instanceof Veiculo); // true (herdado!) System.out.println(c instanceof Caminhao); // false
Saída esperada
true true false
7)
PolimorfismoUpcasting
Intermediário
Demonstre o upcasting criando uma variável do tipo Veiculo que referencia um objeto Carro e mostre que o método sobrescrito é chamado corretamente.
Requisitos
  • Declare Veiculo v = new Carro("BMW", 250, 4) (upcasting)
  • Chame v.exibirInfo() e observe qual versão é executada
  • Tente acessar v.getNumPortas() — o que acontece? (o compilador não permite!)
  • Para acessar métodos específicos de Carro, use downcasting: ((Carro) v).getNumPortas()
Solução
// upcasting — automático e seguro Veiculo v = new Carro("BMW", 250, 4); // polimorfismo: chama Carro.exibirInfo() mesmo com variável Veiculo v.exibirInfo(); // v.getNumPortas(); → ERRO! Veiculo não tem getNumPortas() // downcasting — acessa método específico de Carro int portas = ((Carro) v).getNumPortas(); System.out.println("Portas: " + portas);
Saída esperada
Marca: BMW Vel. máx: 250 km/h Portas: 4 Portas: 4
8)
SobrescreverToString
Intermediário
Adicione em Veiculo (e depois em Carro) a sobrescrita do método toString() herdado de Object. Isso permite usar System.out.println(objeto) diretamente.
Requisitos
  • Em Veiculo, sobrescreva toString() retornando "Veiculo[marca=" + marca + ", vel=" + velocidadeMax + "]"
  • Em Carro, sobrescreva toString() retornando super.toString() + ", portas=" + numPortas
  • Teste com System.out.println(new Carro("Ford", 180, 2))
Em Veiculo
@Override public String toString() { return "Veiculo[marca=" + marca + ", vel=" + velocidadeMax + "]"; }
Em Carro
@Override public String toString() { return super.toString() + ", portas=" + numPortas; }
Saída ao usar println()
Veiculo[marca=Ford, vel=180], portas=2
9)
ListaDeVeiculos
Intermediário
Demonstre o polimorfismo real criando um array do tipo Veiculo[] que armazena tanto Carro quanto Caminhao, e itere chamando exibirInfo() em cada um.
Requisitos
  • Declare Veiculo[] frota = new Veiculo[3]
  • Preencha com: um Carro, um Caminhao e outro Carro
  • Use um loop for para chamar exibirInfo() em cada veículo
  • Observe que cada objeto chama sua versão sobrescrita do método
Solução
Veiculo[] frota = new Veiculo[3]; frota[0] = new Carro("Honda", 200, 4); frota[1] = new Caminhao("Volvo", 120, 15.0); frota[2] = new Carro("BMW", 250, 2); for (int i = 0; i < frota.length; i++) { System.out.println("--- Veículo " + (i + 1) + " ---"); frota[i].exibirInfo(); // polimorfismo! }
Saída esperada
--- Veículo 1 --- Marca: Honda Vel. máx: 200 km/h Portas: 4 --- Veículo 2 --- Marca: Volvo Vel. máx: 120 km/h Carga: 15.0 t --- Veículo 3 --- Marca: BMW Vel. máx: 250 km/h Portas: 2
10)
HierarquiaCompleta
Intermediário
Crie uma hierarquia completa Animal → Cachorro → Labrador, aplicando todos os conceitos da aula: extends, super(), @Override e instanceof.
Requisitos
  • Animal: atributo nome, construtor, método emitirSom() → "..."
  • Cachorro extends Animal: atributo raca, super(nome), @Override emitirSom() → "Au au!"
  • Labrador extends Cachorro: atributo pelagem (String), super(nome, "Labrador"), @Override emitirSom() → "Au au au!"
  • No main: crie um Labrador e verifique com instanceof se ele é Cachorro e Animal
Animal.java
public class Animal { private String nome; public Animal(String nome) { this.nome = nome; } public String getNome() { return nome; } public void emitirSom() { System.out.println("..."); } }
Cachorro.java
public class Cachorro extends Animal { private String raca; public Cachorro(String nome, String raca) { super(nome); this.raca = raca; } @Override public void emitirSom() { System.out.println("Au au!"); } }
Labrador.java
public class Labrador extends Cachorro { private String pelagem; public Labrador(String nome, String pelagem) { super(nome, "Labrador"); this.pelagem = pelagem; } @Override public void emitirSom() { System.out.println("Au au au!"); } }
Main
Labrador lab = new Labrador("Mel", "Dourada"); lab.emitirSom(); System.out.println(lab instanceof Cachorro); // true System.out.println(lab instanceof Animal); // true
Saída esperada
Au au au! true true
