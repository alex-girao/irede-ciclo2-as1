CriarClasseAbstrata
Básico
Crie a classe abstrata Forma com um método abstrato calcularArea() e um método concreto exibirArea().
Requisitos
  • Declare public abstract class Forma
  • Método abstrato: public abstract double calcularArea()
  • Método concreto: exibirArea() que imprime "Área: " + calcularArea()
Solução
public abstract class Forma { // método abstrato — subclasses DEVEM implementar public abstract double calcularArea(); // método concreto — usa polimorfismo internamente public void exibirArea() { System.out.println("Área: " + calcularArea()); } }
2)
ImplementarCirculo
Básico
Crie a classe Circulo que estende Forma e implementa calcularArea() usando π × r².
Requisitos
  • Atributo private double raio
  • Construtor que recebe o raio
  • Implemente calcularArea() com @Override retornando Math.PI * raio * raio
Solução
public class Circulo extends Forma { private double raio; public Circulo(double raio) { this.raio = raio; } @Override public double calcularArea() { return Math.PI * raio * raio; } }
Testando
Circulo c = new Circulo(5.0); c.exibirArea(); // herda Forma.exibirArea() → polimorfismo! // Área: 78.53981633974483

ImplementarRetangulo
Básico
Crie a classe Retangulo que estende Forma e implementa calcularArea() como largura × altura.
Requisitos
  • Atributos private double largura e private double altura
  • Construtor que recebe largura e altura
  • Implemente calcularArea() com @Override retornando largura * altura
Solução

PolimorfismoFormas
Básico
Use polimorfismo para criar um array de Forma com diferentes formas e chamar exibirArea() em loop.
Requisitos
  • Declare Forma[] formas = new Forma[3]
  • Preencha com: um Circulo(5), um Retangulo(4,6) e um Circulo(3)
  • Use for (Forma f : formas) e chame f.exibirArea()
Solução
Forma[] formas = new Forma[3]; formas[0] = new Circulo(5.0); formas[1] = new Retangulo(4.0, 6.0); formas[2] = new Circulo(3.0); for (Forma f : formas) { f.exibirArea(); // despacho dinâmico! }
Saída esperada
Área: 78.53981633974483 Área: 24.0 Área: 28.274333882308138
5)
SobrecarregarMetodo
Básico
Crie a classe Calculadora com três versões sobrecarregadas do método somar() — demonstrando polimorfismo de compile-time.
Requisitos
  • int somar(int a, int b) → retorna a + b
  • double somar(double a, double b) → retorna a + b
  • int somar(int a, int b, int c) → retorna a + b + c
Solução
public class Calculadora { public int somar(int a, int b) { return a + b; } public double somar(double a, double b) { return a + b; } public int somar(int a, int b, int c) { return a + b + c; } }
Testando (compilador escolhe a versão certa)
Calculadora calc = new Calculadora(); System.out.println(calc.somar(2, 3)); // 5 (int) System.out.println(calc.somar(2.5, 3.5)); // 6.0 (double) System.out.println(calc.somar(1, 2, 3)); // 6 (3 ints)

DowncastingSeguro
Intermediário
Escreva um trecho que faz downcasting seguro de uma referência Forma para Circulo usando instanceof antes do cast.
Requisitos
  • Crie Forma f = new Circulo(7.0)
  • Verifique com instanceof Circulo antes de fazer o cast
  • Se for Circulo, faça o cast e acesse o raio via getter (adicione getRaio() em Circulo)
  • Se não for, imprima uma mensagem de aviso
Adicionar em Circulo
public double getRaio() { return raio; }
Downcasting seguro no main
Forma f = new Circulo(7.0); // upcasting automático if (f instanceof Circulo) { Circulo c = (Circulo) f; // downcasting seguro System.out.println("Raio: " + c.getRaio()); } else { System.out.println("Não é um Circulo!"); }

Raio: 7.0

ClasseAbstrataComAtributo
Intermediário
Adicione à classe abstrata Forma um atributo cor (String) com construtor e getter. Atualize as subclasses para usar super(cor).
Requisitos
  • Em Forma: adicione private String cor, construtor e getCor()
  • Em Circulo: construtor recebe raio e cor, chama super(cor)
  • Em Retangulo: similar
  • Atualize exibirArea() para incluir a cor na saída
Forma.java atualizada
public abstract class Forma { private String cor; public Forma(String cor) { this.cor = cor; } public String getCor() { return cor; } public abstract double calcularArea(); public void exibirArea() { System.out.println(cor + ": Área = " + calcularArea()); } }
Circulo.java atualizada
public Circulo(double raio, String cor) { super(cor); // inicializa Forma com a cor this.raio = raio; }
Saída
Circulo c = new Circulo(5.0, "Vermelho"); c.exibirArea(); // Vermelho: Área = 78.53981633974483

SobrescreverToStringForma
Intermediário
Sobrescreva toString() em Forma e em Circulo, demonstrando polimorfismo do método Object.toString().
Requisitos
  • Em Forma: toString() retorna "Forma[cor=" + cor + "]"
  • Em Circulo: toString() retorna super.toString() + ", raio=" + raio
  • Teste usando System.out.println(c) — toString() é chamado implicitamente!
Em Forma
@Override public String toString() { return "Forma[cor=" + cor + "]"; }
Em Circulo
@Override public String toString() { return super.toString() + ", raio=" + raio; }
Saída (toString chamado implicitamente)
Circulo c = new Circulo(5.0, "Azul"); System.out.println(c); // Forma[cor=Azul], raio=5.0

TotalDeAreas
Intermediário
Usando polimorfismo, calcule a soma total das áreas de todas as formas em um array Forma[].
Requisitos
  • Crie um array Forma[] com pelo menos 4 formas (misture Circulo e Retangulo)
  • Use um loop para acumular o total chamando f.calcularArea() em cada forma
  • Imprima o total formatado
Solução
Forma[] formas = { new Circulo(3.0, "Azul"), new Retangulo(4.0, 5.0, "Verde"), new Circulo(2.0, "Vermelho"), new Retangulo(3.0, 3.0, "Amarelo") }; double total = 0; for (Forma f : formas) { total += f.calcularArea(); // polimorfismo! } System.out.printf("Área total: %.2f%n", total);
Saída esperada
Área total: 75.27
10)
HierarquiaCompleta
Intermediário
Adicione uma terceira forma Triangulo à hierarquia sem alterar o código do loop existente — demonstrando a extensibilidade do polimorfismo.
Requisitos
  • Crie Triangulo extends Forma com atributos base e altura
  • Implemente calcularArea(): base * altura / 2
  • Adicione o triângulo ao array Forma[] existente
  • O loop for (Forma f : formas) já funciona — sem nenhuma mudança!
Triangulo.java
public class Triangulo extends Forma { private double base, altura; public Triangulo(double base, double altura, String cor) { super(cor); this.base = base; this.altura = altura; } @Override public double calcularArea() { return base * altura / 2; } }
Main — o loop não mudou nada!
Forma[] formas = { new Circulo(3.0, "Azul"), new Retangulo(4.0, 5.0, "Verde"), new Triangulo(6.0, 4.0, "Rosa") // novo tipo! }; for (Forma f : formas) { f.exibirArea(); // polimorfismo faz o trabalho! }
Saída — Triangulo funciona automaticamente!
Azul: Área = 28.274333882308138 Verde: Área = 20.0 Rosa: Área = 12.0
